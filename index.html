StetMark App Overview

The app showcases a collection of buttons, with each linked to a specific URL. Upon clicking, the URL seamlessly launches within the app's built-in Safari browser. Its purpose is to streamline web access, serving as a handy tool for focused reading or rapidly browsing your top sites. This provides a swift route to repeatedly referenced research articles or cherished content.

Core Features:

URL Buttons:

  • Each button corresponds to a URL.
  • There's a default set of URLs provided, but users can edit them.
  • When a button is tapped, the corresponding URL is opened in an in-app Safari view.
  • The color of the buttons changes based on the neonColors defined.

Edit & Privacy

  • There's an Edit button at the bottom where users can modify the set of URLs.
  • Users can also choose whether the text of each URL is displayed or hidden on the buttons in the main view.
  • The user has the option to enable or disable Face ID/Touch ID authentication for the app.
